# Todo App

A simple and responsive todo app built with React and Vite. It lets you add tasks, mark them as complete, and remove them to keep your daily list tidy.

## Features

- Add new tasks quickly
- Mark tasks as completed
- Delete tasks you no longer need
- Clean, minimal UI

## Tech Stack

- React
- Vite
- Tailwind CSS

## Getting Started

Install dependencies:

```bash
npm install
```

Run the app locally:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
